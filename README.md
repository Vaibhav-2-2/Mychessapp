# Chess Game Application

A real-time multiplayer chess game built with Node.js, Express, Socket.io, and Chess.js. This application allows two players to play chess against each other in real-time, with an interactive user interface for dragging and dropping pieces on the board.

## Features

- Real-time multiplayer functionality using Socket.io
- Drag and drop chess pieces
- Displays board state and moves
- User-friendly interface with Tailwind CSS
- Handles invalid moves and displays appropriate messages
- Support for resigning and resetting the game

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS
- **Backend:** Node.js, Express.js
- **WebSockets:** Socket.io
- **Chess Logic:** chess.js
